# AI Provider Editor

Sanity App SDK workspace package for AI-assisted provider discovery and curation.

## What is included

- `@sanity/sdk` and `@sanity/sdk-react` dependencies
- TypeScript config aligned with the monorepo
- Starter source in `src/index.ts`

## Next steps

1. Add your app UI and SDK integration logic in `src/`.
2. Run `npm install` at the repo root to install dependencies.
3. Run `npm run -w @in-need-of-time/ai-provider-editor type-check`.
